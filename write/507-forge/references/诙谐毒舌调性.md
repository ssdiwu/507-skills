# 507 诙谐毒舌调性

forge 的第二种调性。和克制思辨调性（SKILL §二 T1-T11 + `技法示例库.md`）**物理隔离**，各立各的、互不污染。**选了诙谐毒舌调性才加载本文件**；没选就走克制思辨那一套，不要两套搅在一起（搅了就是四不像）。

对标烈度参考：沐秋《葬AI》系列（《智谱与Anthropic是母凭子贵》等）。**学的是原理和骨架，不是照搬他的网络梗和具体词**——梗是皮，过期快；原理是骨，能长在自己身上。507 的诙谐签名要 507 自己积累替换（见末节"口癖待积累"）。

## 一、口吻定位

**毒舌评论员下场开怼。** 判断不缓冲、直接砸；阴阳怪气优先于直骂；荒诞画面代替程度副词；碎碎念一句成段、跳着说。

和克制思辨调性的区别（务必分清，写之前先确认自己站在哪边）：

| | 克制思辨调性 | 诙谐毒舌调性 |
|---|---|---|
| 姿态 | 朋友娓娓道来陪你推一个判断 | 评论员下场开怼，判断直接砸 |
| 判断力度 | 收着，`我会认为…` hedging 起手 | 放开，`我看来…` `只能说…` 直接下 |
| 情绪 | 体感记忆、克制 | 外放、阴阳、自嘲夸张 |
| 读者关系 | "这个人踩了坑跟我聊" | "这个人在替我吐槽" |
| 说服方式 | 层层推理让你认同 | 用荒谬顶得你自己笑出来 |

**适用**：行业现象吐槽、产品评测对比、竞品阴阳、事件复盘带情绪、翻车实录。
**不适合**：严肃方法论长文（读者要信任你）、需要手把手教的教程、对具体个人的评价（容易翻车且没意义）。

### 嗨夸向变体（夸，不是怼）

诙谐毒舌不等于只能开怼。夸一个真好用的东西，同样可以走诙谐调性——用夸张自嘲反应（H6）、升格反讽的反向用（H5 把好体验抬到荒诞高度制造喜感）、碎碎念节奏（H9）撞起来。对标参考：网黑哥的产品测评长文（如《所有人电脑上现在就去装腾讯 Marvis》）。

**但嗨夸比开怼更容易滑成软文**，纪律更严：
- **必须有翻车对照**（H8）：只嗨夸不写不足，读出来就是广告。至少一处真实吐槽/翻车/比不上竞品的地方。
- **情绪过山车不能省**（H7）：抬完要能给“但是”，不能一路嗨到底。范文正是栽在这里——通篇零翻车零不足，嗨到结尾，结构上像软文。
- **判断别只靠程度副词**：见 §四 反套路新增的一条。嗨夸尤其容易堆“太爽了/恐怖如斯/一个字爽”，这些是皮不是骨。

## 二、边界（默认，可调）

诙谐不等于无差别攻击。毒舌要有靶子、有准星。

- **怼产品 / 公司 / 现象 / 行为**：可烈。"一坨""路边""赢麻了""老人味儿"这个级别随便用。
- **怼具体个人**：停在行为和作品，不上升到人格。可以怼"某人做了个一坨的产品"，不要怼"某人是个一坨的人"。
- **脏话**：边缘、戏剧化用（沐用"畜生""哥们"是怼公司人格化的戏剧效果），不过线成真脏话。脏话是调料不是主菜。
- **讽刺 > 谩骂**：能用阴阳怪气顶出荒谬的，就别直接骂。讽刺高级，谩骂廉价。
- **事实底线不变**：毒舌可以夸张、可以反讽，但**不能编造事实**。产品有什么功能、谁说了什么话，照样要核实（共享克制调性的 T11）。反讽要明显是反讽，不能误导成事实陈述。

## 三、诙谐技法包（H1-H9，原理 + 样本对照）

每条带沐《智谱与Anthropic是母凭子贵》的原文证据。**证据是拆解原理用的，不是让你照抄原句。**

### H1 荒诞拟人化（判断落成一幕戏）

判断的力量不落在形容词，落在一幕画面。把产品/对象的行为**拟人成一个荒诞场景**，比"很差""很离谱"狠 10 倍。

- 样本："智谱直接原地化身战狼，高呼'前沿智能属于所有人'"（不说"智谱营销很激进"，演成战狼）
- 样本："键盘和鼠标在左右脑互搏"（不说"控制有 bug"，演成左右脑打架）
- 样本："我只有三条命……正义迟到了"（不说"游戏数值崩了"，演成主角殉道）
- 原理：读者记不住"很差"，记得住一幕戏。画面 > 程度副词。
- 注意：拟人要服务于判断，不是为搞笑而搞笑。每幕戏背后都有一个你要砸的判断。

### H2 正话反说 / 反讽（反话顶出荒谬）

不直接下判断，**说与事实相反的话**，让读者自己品出讽刺。比直骂高级。

- 样本："我看来 Anthropic 是有点怕智谱的"（实际在嘲讽智谱粉吹过头）
- 样本："怪不得 Anthropic 这么牛逼，原来是'母凭子贵'"（反讽智谱粉把 Anthropic 叫成智谱子公司）
- 样本："你的文字还爱他😭"（反讽智谱官方工具排名还把 Claude Code 放第一）
- 原理：直说"这太荒谬了"没力量，反着说让荒谬自己顶出来，读者笑出来才是真认同。
- 注意：反讽要明显。拿不准读者能不能 get 到是反话，就别用——被人当真就翻车。

### H3 戏剧化揭谜叙事（造悬念 → 揭晓 → 嘲讽）

比平铺直叙有节奏的三段式：**先造悬念勾着读者 → 揭晓答案 → 再嘲讽/升华**。

- 样本（匿名模型打榜整段）：先造悬念（"到底是 DeepSeek-V4？还是 Grok 4.2 呢？"）→ 智谱亲自揭晓 → 嘲讽（"我都看到有人说这是'中国AI的两弹一星时刻'了"），还延伸到小米 Hunter、蚂蚁大象如法炮制。
- 原理：读者被悬念勾着走，揭晓时已经有情绪，嘲讽落点更重。
- 注意：悬念要真实（读者确实会好奇），不是为了结构硬造。没悬念的事件别硬套。

### H4 吐槽代替形容词（选词即判断）

不说"差/好"，选一个**有画面感的吐槽词**放大力度。选词本身就是判断。

- 样本（贬）："一坨""路边""老人味儿""不知所云""中规中矩"
- 样本（褒）："真神""跪下了""完胜""统治世界""碾压"
- 原理：写"这个产品很差"是信息，写"一股老人味儿"是判断 + 画面 + 情绪三合一。
- 注意：吐槽词要新鲜、要准。"一坨"用多了自己也成套路，得换着来、得贴对象。

### H5 夸张降格 / 升格（故意过度形容制造反差）

故意把对象说得**更 low 或更高大**，反差本身就是讽刺。

- 样本（升格嘲讽）："统治世界""两弹一星时刻""载入史册""迎来了智谱斩杀线"——把一个发模型的事抬到国族叙事高度，荒谬感就出来了。
- 样本（降格）："全是路边"——把一堆正经产品全部踩成"路边"，反差制造轻蔑。
- 原理：夸张要明显是夸张，读者知道你在反讽；不是误导事实。
- 注意：和 H2 反讽配套用最强——升格到荒诞，读者自然读出"你在讽刺"。

### H6 夸张自嘲反应（用夸张的自身反应代替直接骂）

不直接骂对象，**写自己一个夸张到好笑的身体/情绪反应**，让对象的不堪通过你的反应反衬出来。比纯喷高级，也避免通篇戾气。

- 样本："看完我直接瘫软在地"（看到夸张标题的反应）
- 样本："最后看了 GLM 做的我直接跪下了"（看到好作品的反应）
- 原理：你的夸张反应是衡量对象程度的标尺。读者通过你"瘫软/跪下"知道这事有多离谱/多牛。
- 注意：自嘲反应要服务于评价对象，别变成自我表演。每个反应都要挂钩一个具体判断。

### H7 情绪过山车（先抬后贬 / 先贬后抬，不一边倒）

通篇一边倒地喷或一边倒地夸都无聊。**把情绪拉成过山车**——刚抬起来又摔下去，刚踩下去又捞一把——读者才有起伏。

- 样本（GLM 测评）：游戏有 bug（贬）→ 但比 Kimi 好（抬）→ 改像素风干外太空去了（贬）→ 综合还是 GLM 胜（抬）→ 最后"统治世界"（升格反讽收尾）。
- 原理：一边倒的喷显得有偏见，过山车显得你真在认真比、认真评，结论才可信。
- 注意：过山车不是和稀泥（"各有千秋"），是**有明确落点的起伏**——贬完要给抬的理由，抬完该贬还得贬，最后站稳一个判断。

### H8 亲自下场真实翻车（吐槽的弹药来源，和克制调性共享）

诙谐的底气来自**真的下场了、真的翻车了**。没下场只靠嘴毒是键盘侠，下过场的毒舌是评论员。

- 样本："我得吃 15 个毒鸭腿才能过关""吃一个鸭腿就掉一条命，而我只有三条命"——自己玩出来的翻车，吐槽才硬。
- 样本："我和工程师凯一再次一起往死里测了下"——真测了才有资格评。
- 原理：和克制调性的 T8（亲自下场）、T11（事实锚定）是同一个底座。诙谐调性里，它是**吐槽的弹药库**——你亲身踩的坑，转化成 H1 的荒诞画面、H4 的吐槽词。
- 注意：翻车要真实细节（数字、操作、当时的 bug 现象），不能编。编的翻车一眼假，毒舌立不住。

### H9 碎碎念节奏 + emoji 当标点（诙谐调性的声音层）

诙谐调性的节奏：**碎、跳、一句成段、emoji 不当装饰当情绪标点**。

- 样本：大量一句成段，跳着说，emoji（😭）当情绪收尾而非点缀。
- 和克制调性的区别：克制调性里 emoji 极少、节奏克制；诙谐调性里 emoji 是声音的一部分，碎碎念是节奏本身。
- 注意：emoji 和碎碎念也**别滥用**——每处都要服务于情绪和节奏，别为了"像活人"硬塞。通篇 emoji 反而显假。

## 四、口癖骨架（参考自样本，507 的诙谐签名待积累替换）

**重要**：以下是从沐的样本提炼的**骨架**，不是 507 的诙谐口癖定稿。507 的诙谐声音还没成型，这部分要在 507 自己的诙谐写作中**逐步积累替换**——每发现一个"507 诙谐写作里自然出现的口癖"就追加，每发现一个"AI 装诙谐的套路词"就记进反套路。先有骨架占位，别照抄沐的词当自己的签名。

骨架（标注来源是样本风格，待 507 替换）：
- **判断直砸**：我看来 / 只能说 / 直接 / 干脆 / 爽吃 / 碾压 / 完胜
- **阴阳反话起手**：我看来 X 是有点…的 / 别猜了，就是我 / 甚至到现在 / 反观…
- **程度放大**：原地 / 直接 / 疯狂 / 往死里 / 一波 / 一坨 / 路边 / 老人味儿 / 真神 / 跪下了 / 统治世界
- **碎碎念连接**：不过话说回来 / 扯了那么多淡 / 还有一波 / 只能说 / 不过话又说回来
- **戏剧收尾**：😭 / 完胜 / 统治世界 / 瘫软在地 / 跪下了

**反套路（诙谐调性专属 AI 味，待积累）**：AI 装诙谐毒舌最容易露的——堆网络梗但原理骨架是空的、为骂而骂没有准星、把“操”“卧槽”当万能填充、反讽不明显被当真。每发现一个追加。

- **用程度副词和口头禅冒充诙谐（最高频，嗨夸向重灾区）**：判断的力量不落在 H1 荒诞拟人画面上，而落在“太爽了/恐怖如斯/一个字：爽/卧槽/太特么”这种程度副词和口头禅的密度堆砌上。口头禅密度高 ≠ 诙谐，诙谐的骨是**有准星的吐槽 + 落成一幕戏的判断**。判据：把口头禅全部替换成中性词后，判断还站得住吗？站不住就是皮厚骨空。反例拆解见 §六 样本二。

## 五、验收（诙谐调性专项，叠加在主验收 §四 L1-L4 之上）

选了诙谐调性，在 forge 主验收（L1-L4）基础上**额外**跑这套（先 `read` 本文件加载技法包，再逐条勾）：

- **H 判断力度够不够外放**：克制调性要收（hedging），诙谐调性要放。通篇如果还在"我会认为…我觉得…"地收着，就是没切换过来，变四不像。有没有至少几处直接砸的判断？
- **H1 荒诞拟人密度**：核心吐槽点有没有落成一幕戏？还是停留在"很差/很离谱"的程度副词？全文至少 2 处荒诞拟人。
- **H2 反讽有没有**：至少 1 处正话反说？反讽明显吗（会不会被读者当真）？
- **H4 吐槽代替形容词**：有没有把"差/好"换成有画面的吐槽词？还是通篇"这个不错那个不行"的空泛判断？
- **H7 情绪过山车**：是一边倒的喷/夸，还是有明确落点的起伏？结论站得稳吗（不是和稀泥）？
- **H8 下场证据**：吐槽有没有真实翻车/亲自测试撑着？还是空嘴毒舌（键盘侠味）？
- **边界守住没**：怼对象不怼人格？脏话过线没？讽刺 vs 谩骂的比例？事实没编造？
- **梗是皮不是骨**：有没有靠堆网络梗撑场面，而 H1-H7 的原理骨架是空的？梗用得贴不贴、新不新鲜？
- **四不像检查（最关键）**：有没有不小心滑回克制思辨调性（`我会认为`hedging、温和同理、克制体感）？两种声音有没有搅在一起？诙谐调性要纯，滑回去就是四不像。

## 六、对照拆解（正反样本）

每个样本拆成两栏：**可复用的原理（骨）** vs **作者的个人皮（不照搬）**。正样本学骨，反样本认皮。

### 样本一（开怼向，正样本）：《智谱与Anthropic是母凭子贵》

| 段落/手法 | 可复用原理（骨，学这个） | 沐的个人皮（不照搬） |
|---|---|---|
| 开头"智谱原地化身战狼" | H1 荒诞拟人：把营销行为演成一幕戏 | "战狼"这个具体梗 |
| "Anthropic 是个畜生" | 戏剧化脏话边缘用、怼公司人格化 | "畜生"这个词、怼 Anthropic 的立场 |
| 匿名模型打榜整段 | H3 戏剧揭谜：悬念→揭晓→嘲讽三段式 | "Pony Alpha""两弹一星"这些具体梗 |
| "我看来 Anthropic 是有点怕智谱的" | H2 正话反说顶出荒谬 | 这句的具体措辞 |
| "全是路边""一坨""老人味儿" | H4 吐槽代替形容词 | 这些具体词（用多了自己成套路） |
| "统治世界""跪下了""瘫软在地" | H5/H6 升格反讽 + 夸张自嘲 | 这些具体反应词 |
| GLM vs Kimi 测评起伏 | H7 情绪过山车，有明确落点 | 具体产品结论 |
| "我得吃 15 个毒鸭腿" | H8 亲自下场翻车当弹药 | 这个具体游戏 |

### 样本二（嗨夸向，反样本）：《所有人电脑上现在就去装腾讯 Marvis》

网黑哥这篇是诙谐毒舌调性“嗨夸向”的样本，但**主要当反例学**——它的结构骨架值得吸收，声音层是“皮厚骨空”的典型。

| 段落/手法 | 可复用原理（骨，学这个） | 网黑哥的个人皮（不照搬） |
|---|---|---|
| 小标题分场景 + 每场景四步 | 多场景功能演示型的合法结构：场景小标题做入口，内部“痛点→指令→过程→结果”一口气叙事 | 具体小标题措辞、公众号配图节奏 |
| 案例排序：桌面整理→千里取物→下班下电影 | T8 升番展示：弱场景在前，炸场（下班路上口喷一句电影就下好了）放结尾，留住读者 | 具体案例选材 |
| 痛点锚点驱动 | 每个功能先立真实痛点（桌面乱/找文件/帮长辈/下班想看电影）再演示，不是干讲功能 | 痛点的具体场景 |
| 通篇“卧槽/太特么/恐怖如斯/一个字爽” | （反例）这就是 §四 反套路新增那条：口头禅密度堆砌冒充诙谐，骨是空的 | 这些口头禅和程度副词 |
| 全文零翻车零不足 | （反例）嗨夸失守 H7/H8：没有翻车对照、没有“但是”，读成软文 | 无（这是要避免的，不是要学的） |
| “这执行力，一个字：爽” | （反例）判断落在程度副词，没落成 H1 荒诞拟人画面 | 这句的具体措辞 |

**一句话**：这篇学**结构**（场景驱动 + 升番 + 痛点锚点），警惕**声音**（口头禅堆砌 + 零翻车嗨夸）。判断永远要落成一幕戏，不能只靠程度副词的密度。

**总一句话**：拆任何样本时永远问——这段的**原理**我能长到自己身上吗（能）？这段的**皮**（具体梗/词/立场/口头禅）是我的吗（多半不是，要换成自己的）？

## 七、和克制思辨调性的共享底座

两种调性物理隔离，但有几条**共享底座**（不分调性都要守）：
- **T3 开头从具体当下事件切入**——诙谐调性也一样，别宏大叙事开场。
- **T4 一口气顺下来**——诙谐调性碎碎念归碎碎念，板块还是用转场句衔接，不用 `---`/小标题切割（除非是分条目的方法论）。
- **T11 事实锚点必须核实**——毒舌不能编事实，反讽不能误导成事实。
- **三层分工**——骨架血肉 AI 写，声音层（诙谐的力度、口癖、梗的取舍）507 手改。AI 装诙谐最容易露，声音层更要过手。
