# 小红书图卡样式预设

`507-rednote` 内置 18 套样式：17 套来自本地 `RedNoteImage` 主题系统，另保留经验证的编辑文档风。

| ID | 名称 | 适合内容 |
|---|---|---|
| `editorial-default` | 编辑文档 | 默认品牌风、技术长文、方法论 |
| `retro` | 复古怀旧 | 历史回顾、个人叙事 |
| `newspaper` | 报纸 | 新闻解读、事实调查、行业进展 |
| `mono` | 极简黑白 | 严肃判断、开发者内容 |
| `nature` | 自然森系 | 温和方法、生活方式 |
| `bluegrad` | 蓝色渐变 | 科技产品、平台能力 |
| `autumn` | 秋日暖阳 | 复盘、经验、故事 |
| `dark` | 深夜暗色 | 深度技术、夜间科技感 |
| `morandi` | 莫兰迪 | 克制思考、个人观察 |
| `cyber` | 赛博朋克 | 强科技感、未来议题 |
| `neubrutalism` | 新野蛮主义 | 强观点、冲突、抓眼封面 |
| `vintage-film` | 胶片复古 | 时间线、旧事复盘 |
| `memphis` | 孟菲斯 | 轻快科普、年轻表达 |
| `editorial` | 杂志排版 | 人物、产品与行业专题 |
| `glass` | 磨砂玻璃 | 产品发布、未来感内容 |
| `bento` | 格子布局 | 模块化方法、产品功能 |
| `y2k` | 千禧复古 | 潮流科技、实验表达 |
| `pink` | 粉色渐变 | 轻内容、生活与情绪表达 |

## 选择纪律

- 同一篇图片文章只使用一套主样式，不逐页随机换皮。
- 样式只改变外观，不得改变 `copy-spec.md` 的文案和论证链。
-用户未指定样式时，先生成封面 + 一页正文的样式联系表，再选择。
- 技术报道默认优先比较 `newspaper`、`editorial`、`mono`、`bluegrad`、`editorial-default`。
- 最终选择不仅看封面，还要看正文长段落是否可读；强视觉主题未必适合长文。

## 自定义颜色

`stylePreset` 确定整体形状、字体与质感；顶层 `theme` 可覆盖 `paper`、`paperAlt`、`ink`、`muted`、`accent`、`accentDark`、`line` 七个颜色变量。只改配色时不新造样式 ID。
