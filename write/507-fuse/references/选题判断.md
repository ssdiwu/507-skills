# 选题判断参考

只在候选方向较多、融合质量拿不准或需要判断“现在值不值得写”时读取；不是每次 fuse 都要举行的打分仪式。

## 三个交集

一个能成立的选题通常同时具备：

- **主体性**：这是作者真正认可、能给出自身判断的内容；
- **读者价值**：读者能获得新事实、新理解或真实共鸣；
- **当下理由**：现在写有具体事件、变化或讨论窗口。

缺一项不必机械淘汰，但候选卡要说明它靠什么补足。只剩热点而没有自身判断，通常不成立；只有专业判断却没有读者入口，需要重新找钩子。

## HKR 辅助检查

- **H（Happy）**：有具体冲突、悬念或反常识，能让人愿意点开；
- **K（Knowledge）**：能给读者新信息或新判断；
- **R（Resonance）**：能连接读者正在经历的困惑或情绪。

至少两项有真实材料承重。不要为了凑齐三项制造夸张标题、假情绪或无关案例。

## 多角色压测

候选难以取舍时，分别问：

- 忙碌的普通读者：它解决了什么具体理解成本？
- 好奇的朋友：最想继续追问的事实是什么？
- 焦虑的学习者：读完会新增哪一个可迁移判断？

三个角色都只能得到泛泛回答，说明创意还没有形成。

## 来源判断

- 一手信息源决定关键事实能否承重；
- 二手聚合源帮助发现线索、争议和写法，但不能替代关键事实；
- 热点需要至少一个可靠事实锚点，不以“多个二手来源都在说”冒充一手证据；
- 既有碎片决定这件事为什么与作者有关，不能只做新闻汇总。
